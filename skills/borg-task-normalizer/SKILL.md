# Skill — Borg Task Normalizer

## Purpose
Ensure all task files follow a consistent structure.

## Responsibilities
- Enforce task template usage
- Standardize task sections
- Normalize headings and metadata

## Required Sections
- Status
- Priority
- Depends On
- Goal
- Inputs
- Scope
- Expected Outputs
- Acceptance Criteria

## Goal
Keep task files predictable and machine-readable.
